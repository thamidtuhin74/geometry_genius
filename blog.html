What is DOM and what is the purpose of DOM?

DOM stands for Document Object Model, that means it break out our document (HTML) into a object tree where every element of our document set as a node. So that we can access & modify our element easily.

DOM gives us a Model or a Layout of the Document so that we can modify our website structure, element & style. DOM helps us to make our website Interactive.

What is event bubble?

Event Bubbling is an important concept in the DOM (Document Object Model). Event Bubbling happens when an element receives an event, and the process starts with the element that triggered the event and then bubbled up to the containing elements in the hierarchy. In event bubbling, the event is captured and handled by the innermost element and then propagated to outer elements.

What is a callback function and why will you use it?

A function called as Callback function when the function is passed into another function as an argument or parameter

Callback Function useful for the asynchronous (objects or events that are not coordinated in time),which means these function(block of code to perform specific task) aren't really stop until they're completed.
